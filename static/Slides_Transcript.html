Slide 4: Before We Start

There are a few terms I need to introduce before continuing.

  1. Braille: a tactile code for an existing language
  2. Braille cell: similar to a 'letter' in English. Combinations of raised dots have different meanings.
    In the example shown, the dots 1,2,3, and 5 create an 'r'.
  3. Unified English Braille (UEB): the current standard for English braille in the USA.
  4. Embossed paper braille: unmoving raised dots written on special paper.
  5. Electronic braille: moveable pins used in devices such as refreshable braille displays.

Slide 13: Conversion: Why Use Shadows and Highlights?

Originally, I only used highlights, or “peaks” (shown in green). However, in poor lighting conditions, false negatives occurred. Some braille dots wouldn't get detected.

I then switched to using only “shadows” (shown in black). This had a similar problem. Some braille dots wouldn't get detected… different braille dots than the ones the peaks missed.

By using both, accuracy of dot detection increases dramatically.
While there are cases where a dot gets picked up “twice”, by both peaks and shadows, this duplicate positive is a lot easier to deal with than a false negative.
In this second image, you can see how the position 2 dot was only detected due to shadow, while dots 1 and 3 were only detected due to peaks, and dot 5 was detected by both.
This diagram on the right goes into more detail about how I used dictionaries to store dot data and determine a dot would begin and end.

Slide 14: Conversion Example

The overall process has three main parts.

First, the original photo. You take that and perform dot detection, which we just talked about.
Once you have the dot coordinates, you turn those into cells. For one cell, this would be like seeing a bunch of pixels on a page and realizing that, together, they make up the letter 'a'.
Using the cells, we use abcbraille.com to finally get the output text.

I like this example because it's short, simple, and also shows the importance of spaces, which were a little difficult to handle. The phrase segment “re be ligh”, part of “Let there be light”, could look like “rebel” and some letters after it, if spaces weren't accounted for.
